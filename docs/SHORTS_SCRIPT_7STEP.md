# 쇼핑 숏폼 대본 7단계 구조 (업계 노하우)

## 7단계

| 단계 | 이름 | 핵심 | 하위 유형 |
|------|------|------|----------|
| ① | 권위 부여 | 신뢰감 확보 | A.전문직업형 B.경력연차형 C.까다로운입맛형 D.유명인형 E.배경스토리형 F.생활베테랑형 |
| ② | 감탄 훅 | 첫 3초 멈추게 | 직접 경험 감정/행동 (간접화법 금지, "난리" 같은 흔한 표현 금지) |
| ③ | 상황 묘사 | 경험 시뮬레이션 | 감각별 묘사 (시각/촉각/미각/후각/청각), 대비 구조, 의태어/의성어 2개+ |
| ④ | 간편함 어필 | 나도 할 수 있겠다 | 숫자 활용("3가지만"), 비밀 재료 숨기기, 진입장벽 낮추기 |
| ⑤ | 과정 묘사 | 하는 장면 (선택) | ③과 겹치면 스킵 가능. 감각 연쇄(시각→후각→신체반응) |
| ⑥ | 사회적 증거 | 주변 반응/본인 변화 | 극단적 전제("안 먹던") + 자발적 행동("먼저 해달라고"), 과대광고 주의 |
| ⑦ | CTA | 행동 유도 | 공감 기반 댓글 유도("시작"/"도전"/"나도"), 링크는 캡션 처리 |

## 고급 기법
- **반전 구조**: 걱정 꺼내고 → 해소
- **연결어미**: 마침표 끊지 않고 ~는데, ~해서, ~하고로 이어가기 (이탈 방지)
- **정보 과밀 주의**: 기능적 장점 최대 2개, 나머지는 감각+반응으로
- **타깃 일관성**: 한 대본에 타깃 2개 이상 금지
- **③+④ 합치기**: 단순 소재는 합쳐도 OK
- **대본 길이**: 30~60초, 군더더기 제거

## 핵심 규칙
- "전문가" 직접 쓰지 않고 직함/경력/행동으로 암시
- 관계어(친구, 엄마, 언니)로 비밀 공유 느낌
- 효능 직접 주장 금지 (과대광고 위험)
- 본인 변화는 간접 체감 표현 ("바지가 헐렁", "피부결이 달라짐")
