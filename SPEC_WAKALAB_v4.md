## 1. 제품 정의

### 한 문장
> 떡상한 커머스 영상을 해킹해서, 내 제품에 맞는 소재를 만들어주는 플랫폼

### 핵심 철학
- 우리는 "이 영상이 팔릴지"를 **판정하지 않는다**
- 우리는 "이 영상이 어떻게 생겼는지"를 **해부해서 보여준다**
- 판단은 사용자가 한다. 우리는 구조를 보여주고, 다음 행동을 도와준다

### 왜 판정하지 않는가
영상의 성과에는 영상 외적 변수(인플루언서 영향력, 플랫폼 알고리즘, 제품 자체 경쟁력, 게시 타이밍, 브랜드 인지도 등)가 영상 내적 요소보다 크게 작용함. 따라서 영상 구조만으로 "팔릴지 안 팔릴지"를 판정하는 것은 불가능하며, 해서도 안 됨.

### 경쟁 포지셔닝
- **Sandcastles.ai** (74K유저, $49~$499/월): "바이럴(조회수) 크리에이터의 리서치+스크립트 도구"
- **WakaLab**: "커머스(전환) 마케터+크리에이터의 해킹+소재 도구"
- 같은 워크플로우(발견→분석→제작), 다른 도메인(바이럴 vs 커머스), 다른 깊이(영상 전체 vs 컷 단위)

## 3. 타겟 사용자

### Segment 1: 쇼핑 숏폼 크리에이터

**프로필:**
- 1인 또는 소규모
- 직접 촬영·편집·업로드
- 수익 = 조회수 x 전환 (쿠팡파트너스 커미션 등)
- 매일 1~3개 영상 제작
- 핵심 고민: "어떤 영상을 만들어야 돈이 되는가?"

**사용 흐름:**
```
레이더 → 해킹 → 대본 → 소재확장
"뭐가 뜨지?" → "어떻게 생겼지?" → "대본 써줘" → "다른 버전도"
```

### Segment 2: 마케터 (에이전시/브랜드)

**프로필:**
- 팀 단위 (마케터 + PD + 편집자)
- 여러 제품 동시 운영
- 성과 보고 필요 (CTR, 조회수 등)
- 영상은 직접 안 만듦, 기획·발주·검수
- 핵심 고민: "경쟁사 대비 우리 영상이 어떤지?"

**사용 흐름:**
```
레이더 → 해킹 → 비교 → 인사이트 → 소재확장
"경쟁사 뭐하지?" → "분석" → "우리랑 비교" → "트렌드" → "다음 소재"
```

## 4. 분석 프레임: 대본 × 영상 2축 구조

숏폼은 크게 **대본(스토리텔링)**과 **영상(완주력)** 두 축으로 구성됨.
- **대본**: 왜 사야 하는가 — 설득력
- **영상**: 끝까지 보게 하는 힘 — 완주력
- **기준점**: 제품/브랜드의 핵심 가치 (product_json에서 추출)

### 4-1. 📝 대본 축: 2단계 분석 (공통 + 스타일 심화)

대본 분석은 **공통 레이어**와 **스타일별 심화 렌즈** 2단계로 구성.

#### 공통 레이어 (모든 영상)
모든 영상에 동일하게 적용. 비교/그룹 분석의 기반. 대본과 영상 양쪽을 포괄.

| 항목 | 축 | 내용 |
|------|-----|------|
| **제품/브랜드 식별** | 기준점 | 제품명, 브랜드, 카테고리, 핵심 가치(USP) — 모든 분석의 기준점 |
| **훅** | 📝 대본 | 첫 3초에 뭘 했는가 — 패턴 + 실제 문구 |
| **소구 포인트** | 📝 대본 | 뭘로 설득하는가 — 유형별 리스트 + claim |
| **설득 흐름** | 📝 대본 | 소구가 어떤 순서로 배치되는가 — 그룹핑 |
| **CTA** | 📝 대본 | 어떻게 마무리하는가 — 유형 + 키워드 |
| **편집 구조** | 🎬 영상 | 어떻게 보여주는가 — 컷 수, 평균 간격, 리듬 패턴 |
| **제품 노출** | 🎬 영상 | 제품을 어떻게 보여주는가 — 노출 비율, 첫 등장, 강조 기법 |

#### 스타일별 심화 렌즈
Phase 0.1에서 분류한 `format × intent` 조합에 따라 자동 선택.

| 스타일 | format × intent | 심화 분석 렌즈 |
|--------|----------------|---------------|
| **크리에이터 리뷰형** | talking_head × commerce | **7요소 프레임워크** (아래 상세) |
| **브랜드 공식 광고형** | brand_ad × brand | 브랜드 아이덴티티, 무드, 스토리 아크 |
| **시연/언박싱형** | product_demo × commerce | 데모 시퀀스, 비포/애프터, 리액션 |
| **캡션/인포형** | caption_text × commerce | 텍스트 구조, 정보 밀도, 가독성 |
| **ASMR/감각형** | asmr × commerce | 사운드 디자인, 감각 자극 시퀀스 |
| **비교/실험형** | comparison × commerce | 비교 구조, 승자 프레이밍, 공정성 |

> **MVP 단계에서는 크리에이터 리뷰형(7요소)부터 구현**, 이후 스타일별 렌즈 확장.

---

#### 심화 렌즈 A: 크리에이터 리뷰형 — 7요소 프레임워크

쇼핑 숏폼 크리에이터의 설득력을 구성하는 **7가지 요소**. AIDA처럼 방법론/체크리스트이며, 순서가 고정된 흐름이 아님.

- 7요소 중 **어떤 것을 사용/미사용**했는가
- 각 요소를 **어떤 순서·비중**으로 배치했는가
- **조합과 배치 자체가 레시피**

예: ①→②→③→⑥→⑦ (④⑤ 스킵) vs ②→④→③→⑥→⑦ (① 없이 훅부터)

해킹 시 각 요소의 사용 여부, 구체적 패턴, 배치 순서를 식별함.

#### ① 권위 부여 (신뢰감 확보)
**핵심:** [신뢰 장치] + [관계성] → "이 사람 말이라면 믿을 수 있겠다"

| 유형 | 예시 | 분석 포인트 |
|------|------|-----------|
| A. 전문직업형 | "합정에서 주방장인 친구" | 직업/직함 + 지역 |
| B. 경력연차형 | "살림 30년 차 우리 엄마" | 경력 수치 + 관계 |
| C. 까다로운입맛형 | "입맛 까다롭기로 소문난 친구" | 극단적 기준 + 관계 |
| D. 유명인형 | "유명 요리 작가가 알려준" | 셀럽 레퍼런스 |
| E. 배경스토리형 | "일본에서 3년 살다 온 언니" | 특정 경험/배경 |
| F. 생활베테랑형 | "아이 셋 키운 베테랑 엄마" | 생활 경험치 |

**분석:** "전문가"를 직접 쓰지 않고 직함/경력/행동으로 전문성 암시하는지, 관계어로 비밀 공유 느낌을 주는지.

#### ② 감탄 훅 (첫 3초 — 멈추게)
**핵심:** 직접 경험한 감정/행동. 간접화법("해외에선 난리") 금지.

| 패턴 | 예시 |
|------|------|
| 직접경험감탄 | "한 입 먹자마자 진짜 말이 안 나오는 거 있죠" |
| 행동변화형 | "이거 먹고 배달 음식 안 시키게 됐어요" |
| 극적반응형 | "바르자마자 피부가 달라지는 거 있죠" |
| 사람반응형 | "시어머니 드렸더니 며느리 주식 올랐어요" |

**분석:** STT 첫 3초에서 훅 문장 추출, 패턴 분류, 직접경험 여부 판단.

#### ③ 상황 묘사 (경험 시뮬레이션)
**핵심:** 감각을 동원해서 "직접 경험하는 느낌". 최소 2~3단계 감각 전환.

| 소재 | 감각 | 묘사 예시 |
|------|------|----------|
| 음식 | 미각+촉각 | "겉은 바삭하면서 속은 폭신폭신 촉촉해서 사르르 녹아내리는데" |
| 청소 | 시각+촉각 | "뿌리자마자 거품이 부글부글 올라오면서 찌든 때가 스르르" |
| 뷰티 | 촉각+시각 | "톡톡 두드리면 결이 정돈되면서 광이 올라오는데" |
| 제품 | 촉각+청각 | "손에 딱 잡히는 순간 묵직한 게 느껴지는데" |

**분석:** STT에서 감각 묘사 구간 추출, 사용된 감각 종류, 의태어/의성어 개수, 대비 구조(겉↔속, 전↔후) 사용 여부.

#### ④ 간편함 어필 (나도 할 수 있겠다)
**핵심:** 진입 장벽 낮추기. 비밀 재료는 숨겨서 궁금증 유지.

| 패턴 | 예시 |
|------|------|
| 숫자 한정 | "재료 3가지만 있으면 돼" |
| 원스텝 | "버튼 하나만 누르면 끝" |
| 시간 한정 | "뿌리고 10분만 놔두면 끝" |
| 공감 | "손재주 없는 나도 하는데" |

**분석:** 숫자 활용("3가지만", "하나만", "10분만") 감지, 비밀 재료 숨김 여부, 복잡한 과정 나열 유무.

#### ⑤ 과정 묘사 (선택 — 해당 시만)
**핵심:** 하는 장면이 보이게. ③과 겹치면 스킵.

**분석:** 데모/조리/사용 장면 유무, 감각 연쇄 흐름(시각→후각→신체반응), ③과 합쳐진 경우 표시.

#### ⑥ 사회적 증거 (주변 반응 / 본인 변화)
**핵심:** 극단적 전제("안 먹던/안 쓰던") + 자발적 행동/대사.

| 유형 | 예시 | 주의 |
|------|------|------|
| 타인반응 | "입 짧은 딸도 먼저 해달라고 조르는" | 자발적 행동 필수 |
| 본인변화 | "한 달 매일 마셨는데 바지가 헐렁해지는" | 간접 체감만 (직접 수치 금지) |

**분석:** 극단적 전제 유무, 자발적 행동/대사 추출, 과대광고 위험 체크(직접 수치/효능 주장 감지).

#### ⑦ CTA (행동 유도)
**핵심:** 공감 기반 댓글 유도. 링크/제품 정보는 캡션에서 처리.

| 패턴 | 예시 |
|------|------|
| 댓글유도 | "다이어트 시작하려는 분들 댓글에 '시작' 남겨주세요" |
| 공감참여 | "피부 고민 있는 분들 댓글에 '공감'" |

**분석:** CTA 유형(댓글유도/링크/구매), 유도 키워드("시작"/"도전"/"나도"), 타깃 일치 여부.

#### 고급 기법 감지

| 기법 | 설명 | 분석 방법 |
|------|------|----------|
| 반전 구조 | 걱정 꺼내고 → 해소 | STT에서 "~아니야?" → "사실은~" 패턴 |
| 연결어미 흐름 | ~는데, ~해서, ~하고로 이어가기 | 마침표 vs 연결어미 비율 |
| 정보 과밀 | 기능적 장점(스펙/성분) 3개 이상 | 기능 소구 개수 카운트 |
| 타깃 일관성 | 한 대본에 타깃 하나 | 시작 타깃 vs 끝 타깃 일치 |

### 4-2. 🎬 영상 축: 완주력 분석

영상이 "끝까지 보게 하는 힘"을 구성하는 요소.

| 분석 항목 | 내용 | 데이터 소스 |
|----------|------|-----------|
| **편집 리듬** | 컷 수, 평균 간격, 패턴(가속/감속/일정/불규칙) | cut_rhythm |
| **시각 에너지** | 화면 변화량 곡선 (edge+color+brightness) | attention_curve |
| **제품 노출** | 노출 비율, 첫 등장 시점, 순환 패턴(사람↔제품) | exposure_curve, product_emphasis |
| **제품 강조 기법** | 클로즈업, 패키지샷, 시연, 질감 등 | emphasis_techniques |
| **사람 활용** | 출연 유형(내레이터/리뷰어), 얼굴 노출, 사람 비율 | presenter, human_time_ratio |
| **자막/텍스트** | 타임라인, 가독성, 텍스트 목적 | caption_map |
| **아트 디렉션** | 톤, 컬러 팔레트, 구도 | art_direction |
| **전환/줌** | 전환 텍스처, 줌 이벤트 | transition_texture, zoom_events |

### 4-3. 🔗 대본 × 영상 교차 분석

7요소 각각에서 **대본이 말한 것 vs 영상이 보여준 것**의 싱크를 검증.

| 단계 | 대본 (STT) | 영상 (프레임) | 체크 |
|------|-----------|-------------|------|
| ① 권위 | "제빵 15년 친구가~" | 주방 배경? 전문가 복장? | 말과 화면 일치 |
| ② 훅 | "한입 먹자마자~" | 먹는 클로즈업? 리액션? | 감탄+시각 동시 자극 |
| ③ 묘사 | "겉바삭 속촉촉~" | 단면 클로즈업? 질감 샷? | 감각 묘사+시각 증거 |
| ④ 간편 | "3가지만 있으면~" | 재료 3개 나열 샷? | 숫자+시각 매칭 |
| ⑥ 반응 | "안 먹던 딸이~" | 아이 먹는 장면? | 증언+영상 증거 |

기존 `appeal_points[].visual_proof`와 `audio_sync` 데이터를 7단계 프레임으로 재구성.

## 5. 해킹 결과: 최종 산출물 = 레시피

모든 분석의 종착점은 **재현 가능한 레시피**.
추상적 라벨이 아니라, 크리에이터가 바로 따라 만들 수 있는 구체적 패턴.

### 레시피 예시

```
[태풍김 · 식품]
```

**목적:** "큰 그림에서 트렌드와 패턴" — 지식그래프 기반

해킹 데이터가 축적되면, 영상 분석 결과를 **구조화된 지식그래프**로 연결:

```

     │              └─ 식품: 경력연차형 45%, 까다로운입맛형 30%
     ├─── 효과적인 훅 패턴
     │              └─ 식품: 직접경험감탄 60%, 행동변화형 25%
     ├─── 주요 묘사 감각
     │              └─ 식품: 미각 80%, 촉각 50%, 후각 30%
     ├─── CTA 패턴
     │              └─ 식품: 댓글유도 70%, 보증형 20%
     └─── 편집 패턴
                    └─ 식품: 평균 2.1초/컷, 감속 패턴 55%
```
