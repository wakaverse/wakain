## T1: 어드민 라우트 + 관리자 인증

- /admin 경로 추가 (App.tsx)
- 관리자 user_id 하드코딩 (주인님 ID)
- AdminLayout 컴포넌트 (좌측 사이드바 + 우측 콘텐츠)
- 비관리자 접근 시 /app으로 리다이렉트

## T2: 📊 대시보드 (첫 화면)

- 기간 선택: 오늘 | 7일 | 30일 | 전체
- 지표 카드 4개: 신규 가입 / 분석 완료 / 활성 사용자 / 재방문율
- 차트 1: 일별 가입자 + 분석 건수 (이중 축 라인)
- 차트 2: 일별 활성 사용자 (막대)
- Supabase stored function으로 집계

## T3: ⚙️ 파이프라인 모니터링

- API 비용 카드 (오늘/주간/월간/건당)
- Phase별 성공/실패율 (가로 막대 차트)
- 평균 분석 시간 추이 (라인 차트)
- 최근 실패 로그 테이블 (20건)

## T4: 👤 사용자 관리

- 사용자 목록 테이블 (이메일/가입일/분석수/플랜/Pro관심)
- 검색 + 필터
- 상세 슬라이드 패널:
  - 기본 정보 + 활동 요약
  - 회수 제한 조정 (JSONB 직접 수정)
  - 분석 결과 목록 (썸네일 + 제목 + 플랫폼 + 날짜)
  - 클릭 시 리포트 페이지 이동
  - 영상 재생 가능 (R2 URL)
  - 최근 활동 로그 30건

## T5: 빠진 이벤트 로깅 추가

- analyze_complete → worker.py 분석 완료 시
- guide_generate → routes/guide.py
- script_generate → routes/guide.py (대본 생성)
- free_limit_reached → services/quota.py 초과 시
