## T2: P13 프롬프트 — 핵심 코칭 중복 제거 + 구간별 일반론 금지
- 핵심 코칭(summary): "영상 요약 반복 금지, 가장 개선 효과가 클 1가지만, 정량 근거 필수"
- 구간별 코칭 금지 패턴 추가:
  - "~하는 것이 중요합니다" / "~하면 효과적입니다" / "~하는 방법입니다" / "~를 높이는 데 효과적입니다"
- 모든 코칭에 필수: 구체적 시간, 정량 데이터, 비교 기준
- 제안은 구체적 행동으로 (시간+블록타입 명시)

## T3: 타임라인 끝부분 정렬
- 모든 레이어(컬러바, 썸네일, 곡선, 스크립트, 소구태그) width를 동일 기준으로
- (time / total_duration) * 100% 통일
- 마지막 구간이 영상 끝까지 정확히 채워지도록

## T4: 씬별 상세 카드
- 타임라인 탭 아래에 씬별 카드 리스트 추가
- 카드 내용: 역할태그+색상, 시간구간, 썸네일, 스크립트 전문, 소구(유형+텍스트+전략), 변화량(평균 대비), 비주얼 스타일
- 재생 버튼 → 해당 시간부터 영상 재생
- 타임라인 클릭 → 해당 카드 스크롤 + 하이라이트
- 카드 시간 클릭 → 타임라인 하이라이트
- 데이터: recipe_json에서 시간 기반 파싱 (analysis_scenes 있으면 사용)

## T5: 요약 탭 구성 정리
