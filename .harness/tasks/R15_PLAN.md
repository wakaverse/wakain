# R15: 리포트 전면 재설계

## 목차 (확정)
1. 영상 요약 — 제품/브랜드/카테고리/타겟/스펙
2. 훅 분석 — 훅 강도+이유, 제품 첫 등장, 제목-훅 정합성
3. 콘텐츠 포지셔닝 — 차별 각도, 스토리텔링 포맷, 대비 구조
4. 제품 소구 분석 — claims 그룹핑+언급 횟수, 노출률
5. 설득 흐름 — 블록 순서 (컬러바 색상만 + 아래 범례), 블록별 내용
6. 타임라인 — 에너지 커브 + 블록 컬러띠 + 이탈 위험 음영 + 씬 경계 점선
7. 코칭 — 잘된점/개선점/추천 구조
8. 씬 분석 — 블록 그룹핑+썸네일, 설득 화법+촬영 유형+claim 연결
